min( a função min() deixa que o css escolha programaticamente o valor menor de uns valores especificados. por exemplo, se você escolher, min(10vw, 500px) o browser faria o elemento de largura 10vw, só enquanto o valor de 10vw seria menor que 500px.)

max( a função max() é o oposto da funcão min(). deixa que o css escolha programaticamente o valor maior de uns valores especificados. se você escrever max(50vh, 200px) o browser faria o elemento de altura 50vh, só enquanto o valor de 50vh fosse maior que 200px.)

calc( a função calc() é uma funcão muito usâvel porque que deixa calcular valores que são dependentes de outros valores. Por exemplo, se você quiser fazer um elemento ao tamanho de meio do viewhight mais um terço do tamanho de uma letra multiplicado por 29 -- por suas razões, não julgo -- você poderia escrever calc(50vh + (1rem / 3 * 29)). obvio, o exemplo não é muito usavel, mas você pode ver como a funcão poderia ser.)

clamp( a função clamp() deixa escolher programaticamente entre valores, mas ao mesmo tempo limitado por um rango. por exemplo, se você quiser ter um elemento ao tamanho do 50vw, mas que não chega maior que 300px ou menor que 100px, você poderia colocar clamp(100px, 50vw, 300px). O elemento só ficaria com 50vw se 50vw estivesse entre 100px e 300px.)

min-width: min(300px, 50vw)
max-width: max(700px, 89vw)
width: calc(var(--valor-px, 180px) + 28vw);
width: clamp(200px, var(--width-preferido, 50vw), 550px);
